## 技术栈

ng5 + bootstrap4 + ngx-toast + scss


## 项目运行

```
git clone https://github.com/alex-0407/ngx-toast-demo

cd ngx-toast-demo  

npm install

npm run dev (运行开发模式)

npm run build (打包发布)


浏览器打开: http://localhost:4200

```
